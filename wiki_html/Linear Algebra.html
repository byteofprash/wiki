Linear spaces

V is a linear space if it follows the following rules:

  • Commutativity
  • Associativity
  • There is an element "0" such that a+0 = a
  • There is an "inverse" element -a such that a+ (-a) = 0
  • 1*a = a
  • λ(μa) = (λμ)a
  • λ(a + b) = λa + λb
  • (λ + μ)a = λa + μa

Subspace of a linear space

U is a subspace V is a linear space A non empty set U ⊂ V is called a subspace if:

  • x,y ∈ U -> x - y ∈ U
  • x ∈ U, λ ∈ R -> λx ∈ U

Notation: U ≤ V

Basis

  • A set of vectors in V is called the basis of V if, it's the generating set and it is linearly independent.

Properties of Basis

  • every linear space V has a Basis
  • Let b₁ , ...bᵢ be the basis of V, then every a ∈ V can be written as a linear combination as a = Σ λᵢbᵢ
  • Every basis in V has exactly n elements in it. This n is called as the dimension of V
